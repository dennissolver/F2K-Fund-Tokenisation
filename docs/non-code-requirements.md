# F2K Non-Code Requirements Checklist

Items outside the codebase that must be completed for the fund to operate.

## Legal & Entity Formation
- [ ] F2K Housing Trustee Pty Ltd incorporated
- [ ] F2K Housing Management Pty Ltd incorporated
- [ ] F2K Operating Pty Ltd incorporated
- [ ] Trust Deed drafted and executed
- [ ] Investment Management Agreement executed
- [ ] Information Memorandum drafted and finalised
- [ ] Convertible note agreement (Tranche 1 seed)
- [ ] Subscription agreement template (Tranche 2)
- [ ] Privacy policy
- [ ] Terms of service
- [ ] Legal opinion on token classification

## Regulatory
- [ ] AFSL application prepared and lodged
- [ ] AFSL granted (or interim authorisation)
- [ ] AUSTRAC DCE registration
- [ ] AML/CTF program documented
- [ ] Compliance arrangements documented (AFSL conditions)
- [ ] Responsible Manager(s) appointed (minimum 2)
- [ ] Digital Assets Framework Bill compliance assessed

## Smart Contract Audit
- [ ] Tier-1 audit firm engaged
- [ ] Audit completed — no critical/high findings
- [ ] Remediation verified by auditor
- [ ] Mainnet deployment post-audit

## Security
- [ ] Penetration test completed
- [ ] Gnosis Safe (production, 3-of-5 real signers)
- [ ] All contract admin roles on Gnosis Safe
- [ ] Deployer key access revoked
- [ ] Bug bounty program established

## Custody & Fund Administration
- [ ] Digital asset custodian selected and onboarded
- [ ] Fund administrator appointed
- [ ] NAV calculation methodology documented
- [ ] Distribution calculation methodology documented
- [ ] Redemption process documented
- [ ] Bank account(s) opened (trustee entity)
- [ ] USDC treasury wallet funded

## Vendor Contracts
- [ ] KYC provider (Sumsub) — production contract signed
- [ ] Custody provider — contract signed
- [ ] Fund administrator — contract signed
- [ ] Market maker — contract signed (Tranche 2)
- [ ] Securitize Markets — listing agreement (Tranche 2)
- [ ] Chainlink — oracle integration

## Insurance
- [ ] Professional indemnity
- [ ] Directors & officers
- [ ] Cyber insurance
- [ ] Crime / fidelity bond

## Board & Team
- [ ] Chair appointed
- [ ] Minimum 4 directors
- [ ] Aboriginal Advisory Sub-Committee formed
- [ ] Audit & Risk Sub-Committee formed
- [ ] Compliance officer (or consultant)
- [ ] Fund accountant (or outsourced)
- [ ] Second Responsible Manager

## Operations
- [ ] Production Supabase project (separate from dev)
- [ ] Production Vercel deployment
- [ ] Production domains configured
- [ ] Monitoring & alerting configured
- [ ] Incident response plan
- [ ] Backup & recovery procedures
- [ ] Operational runbooks reviewed by team
